# IMDb Top 250 Movies — Exploratory Data Analysis

This project performs **Exploratory Data Analysis (EDA)** on the IMDb Top 250 movies dataset to understand patterns in **movie ratings, audience engagement, and release trends**.

## Analysis Overview
- **Univariate Analysis**: Distribution and spread of IMDb ratings and vote counts  
- **Bivariate Analysis**: Relationships between ratings, votes, and release year  
- **Multivariate Analysis**: Combined impact of popularity and release year on ratings  

## Key Insights
- IMDb ratings are **tightly clustered** due to the Top-250 selection bias  
- Vote counts vary widely, reflecting large differences in movie popularity  
- High popularity does not always correspond to higher ratings  

## Tools & Libraries
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  

## Output
- Clean, high-resolution EDA visualizations  
- Saved figures for univariate, bivariate, and multivariate analysis  

---

This project demonstrates a structured EDA approach to understanding how **quality, popularity, and time** interact within IMDb’s highest-rated movies.
